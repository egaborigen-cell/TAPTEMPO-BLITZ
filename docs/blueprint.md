# **App Name**: TapTempo Blitz

## Core Features:

- Rhythm Tap Gameplay: Engage in core gameplay where users tap/click on visual cues as they appear in time with a perceived rhythm.
- Score Tracking & High Score: Display current session score and keep track of the all-time high score for competitive play.
- Dynamic Difficulty Progression: Automatically adjust game speed and the complexity of visual patterns as the player progresses, providing a continuous challenge.
- Game Over & Replay Loop: Present a clear game-over state with the final score and offer an immediate option to restart or try again.
- Adaptive Game Parameter Generation: An AI tool to dynamically generate unique tap patterns and visual sequences, ensuring fresh gameplay with each new session and varied daily challenges.
- Responsive Game UI: Ensure the game interface and controls adapt seamlessly to different screen sizes and orientations for a consistent user experience.

## Style Guidelines:

- Primary color: Vibrant and digital blue (#4DB9E6) to evoke energy and clarity.
- Background color: A serene and clean light blue-grey (#ECF5F8) to provide a non-distracting canvas for gameplay.
- Accent color: An engaging, dynamic purple (#8A73E6) to highlight interactive elements, scores, and important feedback.
- Headline and body font: 'Poppins' (sans-serif) for its modern, geometric, and clear appearance, suitable for a digital game interface.
- Utilize simple, flat, and immediately recognizable icons for all in-game actions like play, pause, retry, and displaying score/settings.
- Implement a centered and uncluttered game area to maintain player focus, with critical information like score and game controls strategically placed for visibility without hindering immersion.
- Incorporate subtle, impactful animations such as glowing feedback on correct taps, quick visual responses to game events, and smooth transitions between different game states to enhance player engagement and feedback.
